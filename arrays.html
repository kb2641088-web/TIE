Java Arrays

An array is a collection of elements of the same data type stored in a single variable. Arrays help you store multiple values without creating many variables.

Example 1: Declaring and Using Arrays


public class ArrayExample {
  public static void main(String[] args) {
    int[] numbers = {10, 20, 30, 40, 50};

    System.out.println("The first number is: " + numbers[0]);
    System.out.println("The last number is: " + numbers[4]);

    // Loop through the array
    for (int i = 0; i < numbers.length; i++) {
      System.out.println("Element " + i + ": " + numbers[i]);
    }
  }
}

Explanation:

  • int[] numbers → declares an array of integers.
  • {10,20,30,40,50} → initializes it with 5 values.
  • numbers.length → gives the total number of elements.
