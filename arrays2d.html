Java 2D Arrays

A 2D Array is like a table or grid — it stores data in rows and columns.

Example: 2D Array of Numbers


public class TwoDArrayExample {
  public static void main(String[] args) {
    int[][] matrix = {
      {1, 2, 3},
      {4, 5, 6},
      {7, 8, 9}
    };

    // Print elements using nested loops
    for (int i = 0; i < matrix.length; i++) {
      for (int j = 0; j < matrix[i].length; j++) {
        System.out.print(matrix[i][j] + " ");
      }
      System.out.println();
    }
  }
}

Explanation:

  • int[][] matrix → creates a 3x3 array.
  • Outer loop → goes through rows.
  • Inner loop → goes through columns.
